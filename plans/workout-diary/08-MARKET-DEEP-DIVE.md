## 1. Конкурентный ландшафт

### Strong (2014, 3M+ users, $4.99/mo)
- UX: ~10 сек / 2-3 тапа на подход
- "What to Beat" — предыдущие значения серым над полями ввода
- Auto-fill из последнего выполнения шаблона
- RPE 6-10, set tags: Warm Up / Failure / Drop Set
- Auto-start timer per-exercise (базовые vs изоляция)
- Supersets через группировку
- Графики e1RM, volume, PR по rep ranges
- Plate calc + warm-up calc = Pro ($4.99/mo, lifetime $99-120)
- **Free tier: 3 шаблона** — основная точка монетизации

### Hevy (2019, 11M+ users, $8.99/mo)
- Самый щедрый free tier: неограниченные тренировки и шаблоны
- ~15 сек на подход (больше UI)
- Цветовые индикаторы PR + немедленное уведомление
- **Smart Superset Scrolling** — автоскролл к следующему упражнению в суперсете
- **Live Activity (iOS Dynamic Island)** — таймер на lockscreen, ±15 сек корректировка, фиксация подходов без разблокировки
- RPE через Settings с улучшенным UI
- **Muscle distribution heatmap** ПРЯМО ВО ВРЕМЯ тренировки
- Социальная лента в стиле Instagram
- Импорт популярных программ (PPL, Starting Strength)

### Fitbod (AI, $12.99/mo — самый дорогой)
- ML-алгоритм генерирует тренировки
- 1-2 тапа при принятии AI рекомендаций
- Brzycki для 1RM, динамический пересчёт
- **Тепловая карта восстановления мышц**
- RIR вместо RPE
- ~50% тренировок с суперсетами
- Мультипрофили залов (дом vs полный зал)
- Warm-up/cooldown одним тапом — 40% юзеров используют

### JEFIT (13M+ users, $69.99/year)
- **2000+ публичных программ**, форумы, челленджи
- Proprietary **NSPI метрика** (North Star Progress Index)
- Критика: нельзя разный отдых для разных подходов одного упражнения

### Нишевые: PRPath, Gainz, FitGo
- PRPath: AI-коуч ATLAS, прогнозные веса по кривой прогрессии + утомление
- Gainz: RP Hypertrophy (Dr. Mike Israetel), set-by-set обратная связь
- FitGo (Узбекистан): 3D-анимации, маркетплейс тренеров, Apple Watch logging

## 2. Quick Logging — анатомия 3-секундного подхода

### Ключевые статистики:
- Логирование ощущаемое как обременительное → **+40% abandonment**
- Золотой стандарт: **1-3 тапа, <5 секунд**
- Тактильная обратная связь → **+30% engagement**

### Паттерны:
1. **"Same as Last Set"** — новый подход = копия предыдущего. 1 тап = готово
2. **"Previous Workout Auto-Fill"** — все подходы из последнего выполнения
3. **Hevy конфиг**: что считать "предыдущим" — last execution вообще vs в рамках программы
4. **StrengthLog rep-PR**: показывает сколько повторений обычно с ЭТИМ весом → если побил число = PR
5. **Числовой пад** > слайдеры/степперы (быстрее в зале)
6. **Тап-зоны ≥44pt** для мокрых рук
7. **Чекмарк = лог + таймер** — одно действие = два результата

## 3. Таймер отдыха

### Адаптивные defaults:
- Базовые (присед, жим, тяга): **120-180 сек**
- Изоляция: **60-90 сек**
- Суперсеты: **0 сек** между упражнениями, отдых после раунда

### UX:
- Inline таймер (не fullscreen overlay!) — сохраняет контекст
- ±15 сек корректировка на ходу (Hevy)
- Live Activity: Dynamic Island + lockscreen + notification shade
- Отдельные настройки громкости: таймер / фиксация / PR (Hevy)
- Smart Superset Scrolling при фиксации в суперсете

## 4. Метрики и формулы

### e1RM формулы:
| Формула | Уравнение | Диапазон |
|---------|-----------|----------|
| Epley | weight × (1 + reps/30) | 2-10, присед/тяга |
| Brzycki | weight × 36 / (37 − reps) | 3-8, жим/присед |
| Mayhew | 100 × weight / (52.2 + 41.9 × e^(-0.055 × reps)) | 3-10, базовые |
| Wathen | 100 × weight / (48.8 + 53.8 × e^(-0.075 × reps)) | 2-10, пауэрлифтинг |

**Рекомендация:** взвешенное среднее нескольких формул. >12 повторений = предупреждение о низкой точности.

### INOL (Intensity × Number of Lifts):
- `INOL_set = reps / (100 − intensity%)`
- Оптимум: **0.8 на упражнение/тренировка**, **2.0/неделя**
- >2.0/тренировка = риск перетренированности

### RPE/RIR:
- RPE 10 = 0 RIR, RPE 8 = 2 RIR, RPE 7 = 3 RIR
- Авторегуляция: **±4% веса на каждый пункт RIR** отклонения

## 5. Plateau Detection

### Четыре метода:
1. **Простой счётчик**: нет PR за ≥3 сессии
2. **SMA comparison**: SMA e1RM за последние 4 тренировки vs предыдущие 4
   - >+1% = прогресс (зелёный)
   - -1% .. +1% = плато (жёлтый)
   - <-1% = регресс (красный)
3. **Линейная регрессия** за 6 недель: нормализованный slope в %/неделя
4. **RPE-чувствительный**: рост RPE при стабильном весе = СКРЫТОЕ плато (усталость)

### Deload:
- Средний цикл: **5.6 ± 2.3 недели**
- Триггеры: запланированный каждые 6 нед, стагнация ≥3 нед, снижение perf, рост RPE
- Рекомендация: **-50% объём**, -10% интенсивность, 7 дней

## 6. Визуализация

- **Линейный график e1RM** per-exercise, тапируемые точки
- **FitNotes**: фильтр по лимиту повторений (10-12 rep) для реалистичных оценок
- **Muscle heatmap на силуэте** (тёмное = часто, светлое = недоработано)
- **GitHub-style calendar heatmap**: размер = duration, цвет = intensity
- **Столбчатая диаграмма** по muscle groups, weekly/monthly/all-time, ←→ навигация
- **Period comparison**: два периода с подсветкой различий
- **Мультимасштаб**: day/week/month/year/all + экспорт как image

## 7. Gamification

### Ключевые данные:
- Achievement notifications → **+65% 90-day retention**
- **70% users churn** в первые 90 дней
- Просмотр данных о прогрессе → **2.3x retention**
- 66% мотивируются сравнением с собой, не с другими

### Паттерны:
- Real-time PR detection + конфетти + haptic
- Отдельный регулятор громкости для PR (Hevy)
- **Streak shield** — защита при болезни/отпуске/отдыхе
- Мультипутевые бейджи: consistency, milestone, strength, volume
- Цветовое кодирование: зелёный = побил, жёлтый = повторил, красный = ниже

## 8. Архитектура (референсы)

### Предрассчитанные агрегаты:
- `daily_exercise_stats`: e1RM, total_volume, max_weight, total_sets per exercise/day
- `weekly_user_stats`: sessions, duration, volume per week
- `weekly_muscle_volume`: sets + volume per muscle group/week
- `user_exercise_prs`: кэш всех PR (обновляется только при новом рекорде)

### Canonical weight storage:
- `actual_weight` (user units) + `actual_weight_kg` (canonical)
- Корректные кросс-единичные сравнения

### Open-source:
- **wger** (Python/Django, PostgreSQL) — FOSS workout manager
- **free-exercise-db** — 800+ упражнений JSON
- **ipalindromi/plate-calculator** — npm, конфигурируемые наборы

## 9. Пять ключевых инсайтов

1. **Базовый UX логирования идентичен у лидеров** — дифференциация в аналитике, социалке, AI

4. **Live Activity для таймера** — новый UX стандарт
5. **Free tier без ограничений** (Hevy, 11M) > ограниченный free (Strong, 3M)
